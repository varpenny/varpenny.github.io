MNMP 环境搭建

2017-09-13 开发 手册 评论

以前在 Win 下开发常用 WAMP (Windows + Apache + MySQL + PHP)，后来换了 Mac 之后用过 XAMPP (各种平台 + Apache + MariaDB + PHP + Perl)，再到现在改用 MNMP (Mac + Nginx + MySQL + PHP)。

Homebrew

Homebrew 是 Mac 上常用的包管理器，Nginx、MySQL 和 PHP 等均可使用它来下载和管理。

Homebrew 安装

Copy 以下命令到 Terminal 执行即可。该命令基于 Ruby，未安装的这边请。

Homebrew 常见概念

  • Formulae（配方）包定义，本质上是一个 Ruby 文件
  • Tap（水龙头）包的来源
  • Keg（桶）包的安装路径
  • Cellar（地窖）包（桶）的根目录
  • Bottle （瓶子）编译打包好的包
    想要更多介绍？这边请

PHP

PHP 安装

这里安装的版本是 PHP5.6，且禁用 Apache，使用 PHP-FPM (FastCGI 进程管理器)。

PHP 配置

PHP 的配置文件为 /usr/local/etc/php/5.6/php.ini，PHP-FPM 的配置文件为 /usr/local/etc/php/5.6/php-fpm.conf，其中 PHP-FPM 配置注意以下几点：

  • 设置 PHP 进程的使用用户，如 user = root 和 group = admin
  • 设置监听 listen = 127.0.0.1:9000

Nginx

Nginx 安装

Nginx 配置

配置文件为 /usr/local/etc/nginx/nginx.conf，注意以下几点：

  • 根据需要修改 user，如 user root admin
  • 确保 user 对于 Nginx 需要 serve 的资源拥有 rwx 权限
  • 修改 error_log、access_log 路径，如 /usr/local/var/log/nginx/error.log
  • 如需设置多站点，建议以 include servers/* 形式引入，并在配置文件目录的 servers 目录下新建对应站点的 config 文件
  • 将 PHP 代码交由 FastCGI 服务处理
